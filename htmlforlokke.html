For-løkker

Løkker er ganske hjelpsom, spesielt om du vil ha samme kode til å gå igjen og igjen, men hver gang har forskjellige verdier.

Her er et eksempel

text += cars[0] + "
";
text += cars[1] + "
";
text += cars[2] + "
";
text += cars[3] + "
";
text += cars[4] + "
";
text += cars[5] + "
";

Istedenfor å repetere arrays på den måten, kan du heller skrive det sånn her. 

for (let i = 0; i < cars.length; i++) {
    text += cars[i] + "
";
}
